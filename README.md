# 🧠 Guardian Module: Compliance Risk Engine

Este repositório contém o **Motor de Decisão** do projeto Guardian. Ele processa dados já normalizados e aplica regras de negócio para classificar o nível de conformidade e risco de uma operação.

### 🚀 Valor de Negócio
O objetivo central deste módulo é a **Automação da Auditoria**. Em vez de um humano analisar cada entrada, o sistema:
- **Filtra Riscos:** Identifica automaticamente níveis de risco (Baixo, Médio, Alto).
- **Escalabilidade:** Permite que a empresa processe milhares de verificações simultâneas sem aumentar o custo operacional.
- **Precisão:** Elimina o erro humano na tomada de decisão preliminar.

### 🛠️ Especificações Técnicas
- **Lógica de Negócio:** Algoritmo de classificação baseado em tokens de segurança.
- **Arquitetura Modular:** Desenvolvido para ser independente da fonte de dados, permitindo fácil manutenção.
- **Foco em Compliance:** Estruturado para atender requisitos de auditoria técnica.
